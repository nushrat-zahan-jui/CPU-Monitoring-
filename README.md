# CPU-Monitoring

Description:
This project is a simple CPU monitoring tool built using Python. The main goal of this project is to display real-time information about the system’s CPU usage in a clean and interactive way.

What I did in this project
In this project, I developed a command-line based CPU monitor where users can choose different options from a menu. I implemented features to show overall CPU usage, display CPU frequency and also added a live monitoring mode that continuously updates CPU usage in real time.

I used the psutil library to collect system information such as CPU percentage and frequency. To make the output more user-friendly, I added color indicators (green, yellow, red) based on CPU usage levels. This makes it easier to quickly understand whether the CPU load is low, moderate, or high.

Technologies Used:
-python 
-psutil library 
-command line interface


Features
-Shows overall CPU usage
-Displays CPU frequency (current and maximum)
-Live CPU usage monitoring
-Color-coded output for better readability


How to Run
-Install Python (if not installed)
-Install required library:
-pip install psutil
-Run the script:
-python CPU.py


This project helped me understand how to work with system-level data in Python and how to create an interactive CLI-based application.
